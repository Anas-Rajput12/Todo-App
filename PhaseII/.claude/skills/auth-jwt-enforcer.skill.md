# Auth JWT Enforcer Skill

## Description
A skill for implementing and enforcing JWT authentication patterns in APIs.

## Capabilities
- Generate JWT authentication middleware
- Validate JWT token formats and claims
- Implement token refresh mechanisms
- Enforce authorization policies
- Generate secure token signing keys

## Usage Examples
- "Add JWT authentication to the todo API"
- "Implement token refresh for user sessions"
- "Validate JWT claims in the authentication middleware"

## Implementation
This skill generates secure JWT authentication code following OAuth2 and JWT best practices.
