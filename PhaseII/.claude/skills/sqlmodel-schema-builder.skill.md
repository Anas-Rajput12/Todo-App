# SQLModel Schema Builder Skill

## Description
Creates SQLModel database schemas with proper relationships, constraints, and validation.

## Capabilities
- Designs database tables with SQLModel
- Implements proper relationships (one-to-one, one-to-many, many-to-many)
- Adds validation and constraints
- Generates migration scripts
- Optimizes database performance

## Usage Examples
- Design user authentication tables
- Create e-commerce product catalog schema
- Build content management system database
- Implement any data model
