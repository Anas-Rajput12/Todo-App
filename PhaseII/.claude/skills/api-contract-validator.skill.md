# API Contract Validator Skill

## Description
A skill for validating API contracts, ensuring they meet specifications and follow best practices.

## Capabilities
- Validate API endpoints against specifications
- Check request/response formats
- Verify authentication requirements
- Validate error handling patterns
- Ensure versioning compliance

## Usage Examples
- "Validate the todo API endpoints against the spec"
- "Check if the authentication API follows JWT standards"
- "Verify error handling in the user management API"

## Implementation
This skill analyzes API definitions (OpenAPI/Swagger) and compares them against specification requirements.
