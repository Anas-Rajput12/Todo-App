# FastAPI CRUD Generator Skill

## Description
Generates FastAPI CRUD endpoints with proper request/response models, error handling, and database integration.

## Capabilities
- Creates complete CRUD endpoints (Create, Read, Update, Delete)
- Generates Pydantic models for request/response validation
- Implements proper error handling and HTTP status codes
- Integrates with SQL databases using SQLAlchemy or SQLModel
- Supports pagination and filtering for list endpoints

## Usage Examples
- Generate user management endpoints
- Create product catalog API
- Build task management system
- Implement any resource-based API
