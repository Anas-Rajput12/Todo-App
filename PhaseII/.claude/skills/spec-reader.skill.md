# Spec Reader Skill

## Description
A skill for reading and interpreting specification documents to extract requirements, constraints, and acceptance criteria.

## Capabilities
- Parse specification documents (Markdown, text, or structured formats)
- Extract key requirements and constraints
- Identify acceptance criteria
- Validate specifications against templates
- Generate summaries of specifications

## Usage Examples
- "Read the todo management spec and extract requirements"
- "Validate this spec against the template"
- "Summarize the authentication requirements from the spec"

## Implementation
This skill uses natural language processing to analyze specification documents and extract structured information.
