# Next.js App Router Builder Skill

## Description
Builds Next.js 16 App Router components with proper routing, data fetching, and state management.

## Capabilities
- Creates App Router page components
- Implements data fetching with Server Components
- Sets up client-side state management
- Configures dynamic routing
- Implements authentication flows

## Usage Examples
- Build dashboard layout
- Create authenticated routes
- Implement data fetching patterns
- Set up complex routing structures
